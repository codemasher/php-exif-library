Make a new entry that can hold an unsigned short.

public __construct(int $tag,  ...$value) : mixed

The method accept several integer arguments. The method will always return an array except for when a single integer argument is given here.

This means that one can conveniently use objects like this: $a = new PelEntryShort(PelTag::EXIF_IMAGE_HEIGHT, 42); $b = $a->getValue() + 314; where the call to will return an integer instead of an array with one integer element, which would then have to be extracted.

Parameters
$tag : int

the tag which this entry represents. This should be one of the constants defined in , e.g., , , or any other tag with format .

$value :

(variadic) the short(s) that this entry will represent. The argument passed must obey the same rules as the argument to , namely that it should be within range of an unsigned short, that is between 0 and 65535 (inclusive). If not, then a will be thrown.
